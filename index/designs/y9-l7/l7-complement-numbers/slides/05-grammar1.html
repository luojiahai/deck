Example Structure Note
老师教得好。 教 + 得 + 好  ·  no explicit object
他滑冰滑得很好。 滑冰 (V+O) → repeat verb 滑 + 得 + 很好
她跑步跑得很快。 跑步 (V+O) → repeat verb 跑 + 得 + 很快
他钢琴弹得很好。 钢琴 (object fronted) + 弹 + 得 + 很好
💡
When the verb has an object (like 滑冰, 钢琴), you have two choices — both are correct!
Option A — repeat the verb:  他滑冰滑得很好
Option B — front the object:  他钢琴弹得很好
