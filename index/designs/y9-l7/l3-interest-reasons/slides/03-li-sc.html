Learning Intention
We are learning to write sentences using 对……感兴趣 and 因为……所以…… to express interest in subjects and give reasons.
1
I can write a correct 对……感兴趣 sentence: Subject + 对 + topic + 很 + 感兴趣 (我对历史很感兴趣。)
2
I can write a complete 因为……所以…… sentence with both clauses in the correct order
3
I can read Text 1 (p.60) and identify how these two structures appear together in context
