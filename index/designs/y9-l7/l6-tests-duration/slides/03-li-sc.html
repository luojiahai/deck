Learning Intention — We Are Learning To …
Ask and answer questions in Chinese about homework, tests, and how long things take (多长时间).
Success Criteria — I Can …
1
Write a duration question and answer using 多长时间
e.g. 要做多长时间？— 大概要做两个小时。
2
Use 功课、测验、考试、学期 correctly in written sentences
3
Write approximate frequency expressions using 每个月……一次 and 每个学期……两三次
