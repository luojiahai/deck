Learning Intention
We are learning to turn a daily-routine statement into a question, and answer it about ourselves or someone else.
1
I can say a daily-routine statement using 我/我们 + time + verb.
用我/我们说日常起居句子
2
I can ask a daily-routine question using 几点.
用几点提问
3
I can describe someone else's routine from a picture using 他/她.
用他/她描述日常起居
