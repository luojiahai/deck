Build-Your-Own Sentence · 自己写句子
Write ONE new sentence (not from Text 2) — a time-of-day word + clock time + a routine verb.
我 + [time-of-day] + [time] + [verb phrase]。
Example
我下午四点半吃早饭。
A silly one is fine for fun — or write a realistic one about yourself!
🚀 Extension: add a second clause with 我们, e.g. 我们八点上课
🗣️ OTR: partner swap — read aloud, thumbs up/down if it makes sense
