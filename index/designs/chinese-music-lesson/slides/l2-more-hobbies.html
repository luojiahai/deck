句型 · Patterns

jù xíng
Using 也 (yě) — also
我也喜欢唱歌。
Wǒ yě xǐhuān chànggē.
I also like singing. (agreeing / adding same hobby)
Using 还 (hái) — in addition
我喜欢唱歌，还喜欢跳舞。
Wǒ xǐhuān chànggē, hái xǐhuān tiàowǔ.
I like singing, and I also like dancing. (adding a different hobby)
Using 都 (dōu) — all / both
我都喜欢！
Wǒ dōu xǐhuān!
I like them all! (used when listing multiple things)
也 = also (same) 还 = additionally (different) 都 = all/both
Teacher: Give prompts — "I like singing. (dancing too)" → 还. "A likes singing. B also likes it" → 也. "I like A, B, and C" → 都喜欢.

Extend · 扩展

kuò zhǎn

Make your answer longer! · 让你的回答更长！

Short → Long

我喜欢唱歌。

↓ add more with 和 / 也 / 还

我喜欢唱歌、弹钢琴和跳舞，还喜欢画画儿。

I like singing, playing piano and dancing, and I also like drawing.

Try it! Start with one hobby, then add more. · 试试看！

Teacher: Call on 4-5 students to share extended sentences. Encourage using 和、也、还 together.

阅读 · Reading

yuè dú

📖 我的爱好

我叫大卫。我有很多爱好。我喜欢弹吉他和画画儿。我也喜欢听音乐。我还喜欢看小说。我觉得看小说很有意思。

Wǒ jiào Dàwèi. Wǒ yǒu hěn duō àihào. Wǒ xǐhuān tán jítā hé huà huàr. Wǒ yě xǐhuān tīng yīnyuè. Wǒ hái xǐhuān kàn xiǎoshuō. Wǒ juéde kàn xiǎoshuō hěn yǒuyìsi.

1.大卫有什么爱好？请列出三件。
2.他觉得什么很有意思？
3.✏️ 你的爱好和大卫一样吗？写一句话。
Students read together, then answer Q1-2 orally. Q3: write in notebook — 3 min individual writing.
