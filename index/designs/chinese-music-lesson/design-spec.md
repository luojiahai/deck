# Design Spec: 轻松学中文2 — Lesson 7 爱好（一）：音乐

## 项目概述
为澳洲 Year 8 中文老师制作的两周课件（8节课 × 50分钟），覆盖《轻松学中文2》Lesson 7「爱好（一）：音乐」。

## 目标受众
- 澳洲 Year 8 学生（13-14岁），中文为第二语言
- 能认读拼音，汉字识字量有限（约100-200字）
- 教室投影仪 1920×1080

## 内容要点（8节课分布）

| 课次 | 主题 | 核心内容 |
|------|------|---------|
| L1 | 认识爱好 | 爱好、音乐、唱歌、听音乐、跳舞；句型「你有什么爱好?」 |
| L2 | 更多爱好 | 弹钢琴、弹吉他、读书、看书、画画；喜欢/不喜欢 |
| L3 | 语法焦点 | 一边……一边……；复合活动表达 |
| L4 | 复习+对话 | 前3课复习；对话练习；中国乐器文化拓展 |
| L5 | 表达原因 | 因为、觉得、有意思；评价爱好 |
| L6 | 时间与频率 | ……个小时、从……开始、正在……、每天 |
| L7 | 阅读与拓展 | 课文阅读、拓展对话、看图说话 |
| L8 | 单元复习+测验 | 总复习、小测验、趣味游戏 |

## 视觉系统

### 色彩
- 主色：暖红 #C43E2C（中国文化联想，但不艳俗）
- 辅色：暖金 #D4953A（点缀用，克制）
- 背景：暖白 #FEFAF5
- 正文：深灰 #2D2A26
- 拼音：中灰 #8B7E74
- 强调区：极淡米色 #F5EDE3

### 字体
- 中文标题：PingFang SC / Microsoft YaHei
- 英文/拼音：system-ui, -apple-system
- 字号层级：标题 64-72px / 正文 36-40px / 拼音 22-24px / 注释 18px

### Layout
- 1920×1080 固定画布
- 大留白、清晰对齐
- 生词卡片：左侧汉字+拼音，右侧英文
- 对话：气泡式或双栏
- 每页信息量克制——投影仪上看不清太密的字

### 反 slop 约束
- ❌ 不用 emoji 做图标
- ❌ 不用紫渐变
- ❌ 不画 SVG 乐器——用 Unsplash 真图或诚实 CSS placeholder
- ❌ 不编造数据
