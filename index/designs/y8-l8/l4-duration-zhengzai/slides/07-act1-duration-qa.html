Activity 1
Written Duration Q&A Building
⏱ 8 min
We Do

Instructions

  1. Teacher gives a sport + duration prompt — e.g. 网球 / 两个小时
  2. Students write the full question-and-answer pair on their whiteboards
  3. Repeat with 3 more prompts covering different verb types (打 / 踢 / no-verb)
  4. Hold up whiteboards after each round for a whole-class check
Sentence Frame 你每天/每个星期几个小时？
我每天/每个星期[duration]。

Model Example

网球/两个小时
你每天打几个小时网球？我每天打两个小时网球。
Scaffold
Sentence frame with blanks stays projected throughout: 你每天/每个星期___几个小时___？我每天/每个星期___[duration]___。
Extension
Students swap 每天 for 每个星期 and adjust the duration accordingly without being told to.
OTR
Whole-class whiteboard hold-up after each prompt.
