èr diǎn
常见错误 #1 — 表示时间，永远用「两」，不用「二」
Mistake #1: When telling the time, 2 o'clock is always 两点 — never 二点.
This applies to 两点、两点半、差十分两点 — all time expressions.
常见错误 · Common Mistake
常见错误 #2–3
Two more patterns the test WILL check — make sure you know these!
Mistake #2: Missing 零
三点五分 ✕
→
三点零五分 ✓
Single-digit minutes after 点 always need 零.
5:05 → 五点零五分 (NOT 五点五分)
Mistake #3: 三刻 means :45
三刻 = :15? ✕
→
三刻 = :45 ✓
一刻 = :15 (one quarter past)
两刻 = :30
三刻 = :45 (three quarters past)
NOT "quarter to" — that's 差!

第二回合 · Answers
常见错误 #4–5
The hardest mistakes — if you can spot these, you are ready.
Mistake #4: 差 word order
七差十分点 ✕
→
差十分七点 ✓
Formula: 差 + minutes + 分 + hour + 点
差 ALWAYS comes first!
差 = "to" (as in "ten to seven")
Mistake #5: 六十 rolls over
十二点六十 ✕
→
一点 / 十三点 ✓
60 minutes = next hour.
There is no 六十 on a clock —
12:60 is really 1:00!
Think of it as a conceptual error, not just a Chinese error.
