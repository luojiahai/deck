L6 · ASKING TIME

Practice · Chain Dialogue

18–25 min · Activity 1

Chain Dialogue · 对话链

Whole-class chain. Ask the person next to you, they answer, then they ask the next person!

Whole Class

You

Ask →

Neighbour

Answers →

Neighbour

Ask →

Next

1

No repeating times. Each person must give a different time from everyone before them.

2

No same pattern twice in a row. If the last person said 半, you must use a different pattern (刻, 分, 差...).

3

Listen carefully. You need to know what patterns have already been used!

4

Keep it moving. Max 10 seconds per person. If stuck, the class can help.

Example

A: 现在几点？ B: 三点一刻。 (turns to C) C: 差五分六点。
