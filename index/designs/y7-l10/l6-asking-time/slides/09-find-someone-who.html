L6 · ASKING THE TIME Practice · Find Someone Who
33–43 min · Activity 3
Find Someone Who… · 找人
Walk around the room. Find a different person for each prompt. When you find someone, write their name!
1
找一个人，他的时间用「半」。
Find someone whose time uses 半 (half past).
Pattern: 半
Name: ________
2
找一个人，他的时间用「差」。
Find someone whose time uses 差 (minutes to).
Pattern: 差
Name: ________
3
找一个人，他的时间用「刻」。
Find someone whose time uses 刻 (quarter).
Pattern: 刻
Name: ________
4
找一个人，他的时间里有「零」。
Find someone whose time uses 零 (e.g. X:05).
Pattern: 零
Name: ________
5
找一个人，他的时间超过9 点。
Find someone whose time is after 9:00.
Time > 9:00
Name: ________
6
找一个人，用「请问」问他/她时间。
Find someone and ask them using 请问 politely.
Polite: 请问
Name: ________
📣 Rules: Different person for each prompt (6 different classmates). Must ask「现在几点？」in Chinese. Listen to the pattern they use, not just the time.
