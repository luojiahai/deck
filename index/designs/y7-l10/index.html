Courseware · 课件

时间 · Telling the Time

轻松学中文 · Time Unit | Year 7 Chinese · 七年级中文 | 8 lessons × 50 min | Est. ~120 slides

Pattern Progression · 句型递进

L1 · X + 点 → L2 · X点Y分 → L3 · X点半 → L4 · X点Y刻 → L5 · 差X分Y点 → L6 · 现在几点? → L7 · Mixed Practice → L8 · Review + Test

Lessons · 课程

8 lessons · ~120 slides total
Lesson 1

几点？

Jǐ diǎn? · What's the Time? Numbers review + o'clock（点）— textbook p84/p87 warm-up, key rule 两点≠二点, clock flash, telephone game, exit ticket
16 slides 🔢 Numbers 🕐 点 🎮 Game
Lesson 2

几点几分？

Jǐ diǎn jǐ fēn? · Minutes Adding minutes（分）— X点Y分 pattern, 零五 rule, clock drawing, listening bingo
~16 slides 🕐 分 🎧 Listening
Lesson 3

几点半？

Jǐ diǎn bàn? · Half Past Half past（半）— X点半 = X:30, clock hands at half-past position, pair speaking practice
~15 slides 🕐 半 🗣️ Speaking
Lesson 4

几点几刻？

Jǐ diǎn jǐ kè? · Quarters Quarters（刻）— 一刻:15 / 三刻:45, rule 三刻≠四刻, clock bingo, all patterns so far
~15 slides 🕐 刻 🎮 Bingo
Lesson 5

差几分？

Chà jǐ fēn? · Minutes To Minutes to（差）— 差X分Y点 = X to Y, visual clock arcs, train schedule reading
~16 slides 🕐 差 🚂 Reading
Lesson 6

现在几点？

Xiànzài jǐ diǎn? · Asking the Time Conversation — 现在几点? Q&A with ALL 5 patterns, pair dialogues, daily schedule speaking
~16 slides 💬 Dialogue 🗣️ Speaking
Lesson 7

综合练习

Zōnghé liànxí · Mixed Practice All 5 patterns mixed — random clock flash, listening dictation, daily schedule writing, pair Q&A
~16 slides 🔄 Mixed ✏️ Writing
Lesson 8

复习与测验

Fùxí yǔ cèyàn · Review + Test Full review + practice test — common mistakes, listening/reading/writing sections, final game 🎉
~18 slides 📝 Test 🔍 Review 🎉
