Project the reference grid for 1 minute of final study — then hide it for the 3-minute write phase.

How to Play

1
3–4 teams; one scribe writes for the team (or each member writes and the team's best set counts)
2
Study grid 1 min → grid hidden
3
3 min: write as many radicals as remembered, each with one example word
4
Teams read list aloud — 1 pt per correct radical + 1 bonus pt if the example word is also correct. Track on whiteboard corner.

OTR 1

Every team member must contribute at least one radical — teacher circulates during the 3-minute window to confirm.

Scaffold

Teams may keep the meaning column (English) visible as a memory hook — only 汉字/example columns hidden.

Extension

Bonus point for any radical beyond the twelve shown (e.g. 讠, 刂, 辶) if correctly identified with an example.
