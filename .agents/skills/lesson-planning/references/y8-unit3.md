### Key Sentence Patterns

### Key Texts

**Text 1 (p. 60, CD Track 31)**
> 我叫田云。我有很多爱好。我喜欢一边弹钢琴，一边唱歌。我还喜欢听音乐和读书。我每天读一个小时的书。

**Text 2 (p. 64, CD Track 34)**

> B: 我喜欢看小说和杂志。
> A: 你喜欢画画儿吗？
> B: 喜欢。油画儿、水彩画儿，我都喜欢画。我正在学画国画儿。
> A: 我也在学画国画儿。

### Grammar Notes

- **一边……一边……** connects two simultaneous actions by the SAME subject (one person doing two things at once)
- **Duration structure**: 每天 + verb + number + 小时/分钟 + 的 + noun (e.g. 他每天打一个小时的电话)
- **正在 + verb** = action in progress (present continuous)
- **……，都……** = both/all — list the items BEFORE 都, and 都 goes before the verb (e.g. 油画儿、水彩画儿，我都喜欢)

### Key Sentence Patterns

- 我喜欢跟朋友一起打网球。(with someone)
- 我不太喜欢运动。(not very/really)

- Sports verb rules: 打 + ball sports (篮球, 乒乓球, 网球, 羽毛球, 高尔夫球); 踢 + 足球; no verb prefix for 滑雪, 滑冰, 跑步, 游泳

### Key Texts

**Text 1 (p. 70, CD Track 36)**
> 我叫高力。我们一家人都喜欢运动。我爸爸喜欢跑步。他每天跑一个小时步。我妈妈喜欢游泳。她每天早上游泳。我呢，我喜欢跟朋友一起打网球。

**Text 2 (p. 74, CD Track 39)**
> A: 我不太喜欢运动，但是我喜欢游泳。你喜欢运动吗？
> B: 我很喜欢运动。我还喜欢看电视、看电影。
> A: 你喜欢看什么电视节目？
> B: 我喜欢看体育节目。
> A: 你经常看电影吗？
> B: 我差不多每个周末都看。

### Grammar Notes

- **跟(和)……一起** = together with
- **不太 + adjective/verb** = not very / not really
- **差不多** goes before the main clause: 我差不多每个周末都看
- **经常/常常** goes before the verb (adverb position)
- **Sports verb rules**: 打 for hand/racket ball sports; 踢 for football (kicking); no verb prefix for 滑雪, 滑冰, 跑步, 游泳

### Key Sentence Patterns

- 除了音乐以外，我还喜欢舞蹈。(besides X, also Y)
- 我从四岁开始学跳舞。(starting from age X) — NOTE: 四岁, not 五岁

### Key Texts

**Text 1 (p. 80, CD Track 41)**

**Text 2 (p. 84, CD Track 44)**
> A: 你好！大生在家吗？
> B: 我就是。
> A: 你在干什么呢？
> B: 我在一边做作业一边玩儿电脑游戏呢。
> A: 我们一起去打篮球吧！
> B: 好。什么时候去？
> A: 下午四点，好吗？
> B: 可以。我们四点在球场上见！
> A: 好吧。一会儿见！

### Grammar Notes

- **除了X以外，还Y** = besides X, also Y
- **从X开始** = starting from X (age, time, place)
- **Instrument verbs**: 拉 for bowed strings (violin, erhu); 弹 for keyboard/plucked (piano, guitar)
- **跳 + 舞** (dance); **玩 + 游戏** (play games)
- **(正)在……呢** = action continuing right now (stronger than 正在 alone)
- **吧** = suggestion particle (我们一起去游泳吧)
- **什么时候** (when — any time frame) vs **几点** (what time — hour only)

## Workbook Exercises (轻松学中文 2 Workbook)

*Workbook 2, Unit 3 (book pp. 72–107). Maps the homework/practice formats and content that accompany the textbook. Reference-style — grouped by exercise type, not exhaustive.*

### Exercise formats (recurring across all three lessons)

- **Character writing**: trace/write each lesson's New Words; copy the 4 simple characters (confirms the textbook list)
- **Vocab & phrases**: verb–noun matching, fill-missing-word, word-search grids, radical/structure work, dictionary look-ups (write the English meaning)
- **Grammar/sentence**: production with the unit's patterns (一边……一边…… / 跟……一起 / 但是 / 除了……以外 / 从……开始 / 正在 / 吧), duration "verb + 时间 + 的 + object", sentence rearrangement, error-correction
- **Translation**: both E→C and C→E (durations, language ability, 正在, advice)
- **Reading**: comprehension passages + cloze (family hobbies, daily routines)
- **Writing**: ~10 personal-answer questions; a guided essay (reply letter / account of a visit)

### Per-lesson highlights

**L7 Music (WB pp. 72–81, 20 ex.)** — duration translation E→C (半个小时 / 一刻钟 / 三刻钟 …); verb–noun match (穿/采/说/上/做/读/唱/弹/看); 一边……一边…… / 有时候 / 经常; 正在 (9 pictures); dictionary look-up (彩色/豆油/手风琴/口琴/午休/左边/书包/儿歌/米饭); error-correction (word order, e.g. 一边 misplacement); essay = reply letter (family, languages, family hobbies).

## Unit 3 Cross-Cutting Notes

### Skills Progression

1. **Lesson 7** → Music hobbies + 一边……一边…… (simultaneous actions) + art vocabulary + 都 (both/all) + 正在 (present continuous)
2. **Lesson 8** → Sports hobbies + 差不多/经常/常常 (frequency adverbs) + 跟……一起 (with someone) + 不太 (not very)
3. **Lesson 9** → Dance/performance + 除了……以外 (besides) + 从……开始 (starting from) + 呢/吧 particles + 什么时候 vs 几点

### Common Student Errors

- **Sports verbs**: Using 打 for all sports — correct: 踢足球 NOT 打足球; no verb prefix for 跑步, 游泳, 滑雪, 滑冰
- **Confusing 一边……一边……** (one person doing two things simultaneously) with **跟……一起** (two people doing one thing together)
- **Instrument verbs**: 拉 for bowed strings (violin, erhu); 弹 for keyboard/plucked (piano, guitar) — not interchangeable
- **Adverb placement**: 常常/经常/差不多 go BEFORE the verb, not after
- **都 placement**: after subject, before verb — not at the end of the sentence (油画儿、水彩画儿，我**都**喜欢 ✓; NOT 我喜欢都 ✗)
- **四岁 in L9 Text 1**: The subject starts dancing at 四岁 (age 4), not 五岁 — a common error from earlier incorrect versions
- **什么时候 vs 几点**: 什么时候 = when (any time frame — morning, weekend, which day); 几点 = what time (hour only — 几点上课？)
