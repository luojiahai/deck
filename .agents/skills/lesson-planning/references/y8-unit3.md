## Lesson 7: Hobby (1) — Music (爱好(一): 音乐)

### Core Vocabulary

| 汉字 | 拼音 | English |
|---|---|---|
| 爱好 | àihào | hobby |
| 一边…一边… | yìbiān…yìbiān… | at the same time; doing X while doing Y |
| 弹 | tán | play (a stringed musical instrument) |
| 钢琴 | gāngqín | piano |
| 弹钢琴 | tán gāngqín | play the piano |
| 吉他 | jítā | guitar |
| 弹吉他 | tán jítā | play the guitar |
| 唱 | chàng | sing |
| 歌 | gē | song |
| 唱歌 | chànggē | sing a song |
| 听 | tīng | listen |
| 音乐 | yīnyuè | music |
| 听音乐 | tīng yīnyuè | listen to music |
| 读 | dú | read; attend |
| 书 | shū | book |
| 读书 | dúshū | read a book; study |
| 小时 | xiǎoshí | hour |
| 小说 | xiǎoshuō | novel |
| 杂志 | zázhì | magazine |
| 报纸 | bàozhǐ | newspaper |
| 都 | dōu | all; both |
| 正在 | zhèngzài | in the process of (present continuous marker) |
| 还 | hái | also; still |

### Key Sentence Patterns

**Expressing simultaneous actions:**
- 我一边弹钢琴，一边唱歌。(I sing while playing the piano.)
- 他一边吃饭一边看电视。(He watches TV while eating.)

**Expressing hobbies:**
- 我喜欢弹钢琴。(I like playing the piano.)
- 我还喜欢听音乐和读书。(I also like listening to music and reading.)

**Duration:**
- 我每天读一个小时的书。(I read for one hour every day.)
- 你每天看几个小时的书？(How many hours do you read each day?)

**Present continuous with 正在:**
- 你正在听什么音乐？(What music are you listening to?)

### Key Text (Textbook p64)
```
我叫田云。我有很多爱好。我喜欢一边弹钢琴，一边唱歌。我还喜欢听音乐和读书。我每天读一个小时的书。
```
Translation: My name is Tian Yun. I have many hobbies. I like singing while playing the piano. I also like listening to music and reading. I read for one hour every day.

### Grammar Notes
- 一边…一边… connects two simultaneous actions by the same subject
- 小时 is the unit for hours of duration (not 点, which is for o'clock)
- 都 (dōu) means "all" or "both"
- 正在 + verb = action in progress (like present continuous)

### Test Coverage (from y8-tests/)
- **Test 3.1**: Translation sentences (一边…一边… pattern, duration phrases); vocabulary table: hobbies, music terms, time units
- **Test 3.2**: Translation with adverbs (差不多, 常常, 跟…一起); sports vocabulary table
- **Test 3.3**: Question answering about TV/eating habits; dance/gaming vocabulary; time expressions

| 跟…一起 | gēn…yìqǐ | together with |

| 常常 | chángcháng | often |
| 通常 | tōngcháng | usually |
| 每 | měi | every |

| 每年 | měi nián | every year |

| 但(是) | dàn(shì) | but; however |
| 不太 | bú tài | not too; not very |

### Key Sentence Patterns

**Talking about sports:**
- 我喜欢打篮球。(I like playing basketball.)
- 我不太喜欢打篮球。(I don't really like playing basketball.)
- 我差不多每天都去滑冰。(I go ice skating almost every day.)

**Playing sports — 打 vs others:**
- 打 + ball sport: 打篮球, 打乒乓球, 打网球, 打羽毛球
- 踢 + 足球 (kick football)
- No 打: 滑雪 (ski), 滑冰 (ice skate), 跑步 (run), 游泳 (swim)

**With someone:**
- 我跟Jerry一起去打乒乓球。(I play table tennis with Jerry.)
- Tom跟Jerry一起去打乒乓球，但是Tom打得不太好。(Tom plays table tennis with Jerry, but Tom doesn't play very well.)

### Key Text
About sports hobbies — students describe which sports they like, how often they do them, and who they do them with.

### Grammar Notes
- 打 (dǎ) is used for ball sports you hit with hand/racket
- 踢 (tī) is used for football/soccer (kicking sports)
- No verb prefix for activities like 滑雪, 滑冰, 跑步, 游泳
- 跟…一起 = together with (跟 marks the companion)
- Adverbs (常常, 差不多, 通常) go before the verb

### Test Coverage (from y8-tests/)
- **Test 3.2**: Translation of sentences with adverbs; sports vocab; measure words and time expressions

## Lesson 9: Hobby (3) — Dance (爱好(三): 跳舞)

### Core Vocabulary

| 汉字 | 拼音 | English |
|---|---|---|
| 跳舞 | tiàowǔ | dance |
| 舞蹈 | wǔdǎo | dance (formal) |
| 游戏 | yóuxì | game |
| 电脑游戏 | diànnǎo yóuxì | computer game |
| 玩 | wán | play (games) |
| 功课 | gōngkè | homework; schoolwork |
| 作业 | zuòyè | homework; assignment |
| 做作业 | zuò zuòyè | do homework |
| 什么时候 | shénme shíhou | when; what time |
| 干什么 | gàn shénme | do what |
| 时候 | shíhou | time; moment |
| 一会儿 | yíhuìr | a little while |
| 除了…以外 | chúle…yǐwài | besides; apart from |
| 从 | cóng | from |
| 开始 | kāishǐ | start; begin |
| 拉 | lā | pull; play (certain string instruments) |
| 小提琴 | xiǎotíqín | violin |
| 拉小提琴 | lā xiǎotíqín | play the violin |

### Key Sentence Patterns

**Dance and movement hobbies:**
- 我从X岁开始学跳舞。(I started learning dance at age X.)
- 我每个星期二、星期四跳舞。(I dance every Tuesday and Thursday.)

**Asking about activities:**
- 你晚上看什么电视节目？(What TV programs do you watch in the evening?)
- 你看几个小时的电视？(How many hours of TV do you watch?)
- 你几点睡觉？(What time do you go to bed?)

**Beyond one hobby — 除了…以外:**
- 除了音乐以外，我还喜欢跳舞。(Besides music, I also like dancing.)

**Talking about routine activities:**
- 我每天拉四十五分钟小提琴。(I play the violin for 45 minutes every day.)

### Key Text (Textbook ~p84)
```
你好！我叫欢欢。除了音乐以外，我还喜欢跳舞。我从五岁开始学跳舞。我每个星期二、星期四跳舞。我哥哥从六岁开始拉小提琴。他每天拉四十五分钟小提琴。
```

### Grammar Notes
- 除了X以外，还Y = besides X, also Y
- 从X开始 = starting from X (age, time, etc.)
- 拉 is used for bowed string instruments (violin, etc.), 弹 for plucked/keyboard (piano, guitar)
- 跳 + 舞 (dance), 玩 + 游戏 (play games)

### Test Coverage (from y8-tests/)
- **Test 3.3**: TV/activity questions; dance vocabulary; time expressions; 什么时候, 干什么

## Unit 3 Cross-Cutting Notes

### Skills Progression
1. Lesson 7 → Music hobbies + 一边…一边… (simultaneous actions)
2. Lesson 8 → Sports hobbies + adverbs of frequency + 跟…一起
3. Lesson 9 → Dance/performance hobbies + 除了…以外 + 从…开始

### Common Student Errors
- Using 打 for all sports (not for 跑步, 游泳, 滑雪, 滑冰)
- Confusing 一边…一边… (same person, two actions) with 跟…一起 (two people, one action)
- Word order with adverbs: adverb goes BEFORE the verb
- Forgetting 都 after listing items: 我喜欢弹钢琴和唱歌。→ No 都 needed; 我都喜欢弹钢琴和唱歌 → 都 goes before verb

### Textbook Activities & Radicals

Like all 轻松学中文 units, Unit 3 includes:
- **Radicals**: Each lesson has a "Learn the radicals" section. Identify which radicals appear on each page and teach them alongside the relevant characters (e.g. 扌(hand) → 打, 拍; 氵(water) → 游泳, 池).
- **Speaking practice**: Dialogue/model-sentence exercises throughout the unit.
- **Listening exercises**: Listen and tick/circle/answer questions.
- **Pinyin exercises**: Pinyin discrimination and tonal mark practice.
- **Group/inquiry activities**: Dictionary look-up races, category word games.

When planning lessons for this unit, check the textbook pages for the specific radicals, pinyin pairs, and activity formats — cover them all across the lesson sequence, the same way Y7 Lesson 10's 8-lesson sequence covers every page.
