## Lesson 10: Time (时间)

### Core Vocabulary

| 汉字 | 拼音 | English |
|---|---|---|
| 点 | diǎn | o'clock |
| 分 | fēn | minute |
| 刻 | kè | quarter (of an hour) |
| 半 | bàn | half |
| 两 | liǎng | two (used before 点) |
| 零 | líng | zero |
| 现在 | xiànzài | now |
| 差 | chà | fall short of; to (time) |
| 表 | biǎo | watch |

### Key Sentence Patterns

**Asking the time:**
- 现在几点？(What time is it now?)

**Telling the time:**
- X点Y分 — e.g. 三点十分 (3:10)
- X点半 — e.g. 两点半 (2:30)
- X点一刻 — e.g. 九点一刻 (9:15)
- X点三刻 — e.g. 四点三刻 (4:45)
- 差Y分X点 — e.g. 差十分七点 (10 to 7 / 6:50)

**Core dialogue (Textbook p89):**
- A: 现在几点？
- B: 差十分九点。

### Grammar Notes
- 两 vs 二: 两点 (2:00) NOT 二点
- 差 means "to" in time expressions: 差X分Y点 = X minutes to Y o'clock
- Time expressions can go at the beginning or after the subject
- 零 is required before single-digit minutes (三点零五分, not 三点五分)

### Radicals (Textbook p85, p93, p101)

Lesson 10 introduces 6 radicals. Each is linked to a character in this unit:

| 部首 | Name | Meaning | Related character | When to teach |
|---|---|---|---|---|
| 土 | tǔ | soil; earth | 地 (dì — ground) → 地铁 | With 分/零 (Lesson 2 of 8) |
| 寸 | cùn | unit; inch | 时 (time, trad. 時) → 时间 | With 分/零 (Lesson 2) |
| 刂 | dāo (立刀旁) | knife | **刻** (亥 + 刂 = 刻) | With 刻 (Lesson 4) |
| 米 | mǐ (米字旁) | rice | 粒, 粮, 糖 — common food chars | With 刻 (Lesson 4) |
| 彡 | sān (三撇旁) | ornament | 形, 影, 彩 | With 表 (Lesson 5) |
| 衤 | yī (衣字旁) | clothing | **表** (bottom component from 衣) | With 表 (Lesson 5) |

**Why these radicals matter:** 刻 contains 刂 (knife cutting time into quarters). 表 contains 衤 (a watch is something you wear — clothing radical). Teaching radicals helps students analyse and remember characters rather than rote-memorise them.

### Pinyin Discrimination (Textbook p89)

Students must distinguish these common pinyin confusions (all appear in Lesson 10):

| Pair | Correct | Common Error |
|---|---|---|
| shén vs sén | shén (什) | sén (non-existent) |
| nǐ vs ní | nǐ (你, 3rd tone) | ní (incorrect tone) |
| mèi vs měi | mèi (妹, 4th tone) | měi (每, 3rd tone — different word) |
| qī vs chī | qī (七) | chī (吃 — different word!) |
| zhè vs zhé | zhè (这, 4th tone) | zhé (incorrect tone) |
| zài vs zhài | zài (在, 4th tone) | zhài (non-existent) |
| hào vs hòu | hào (号) | hòu (non-existent) |
| bàn vs bìng | bàn (半) | bìng (non-existent) |
| guó vs guǒ | guó (国, 2nd tone) | guǒ (果 — different word) |

## Lesson 11: Daily Routine (日常起居)

### Core Vocabulary

| 汉字 | 拼音 | English |
|---|---|---|
| 早上 | zǎoshang | (early) morning |
| 上午 | shàngwǔ | before noon; morning |
| 中午 | zhōngwǔ | noon |
| 下午 | xiàwǔ | afternoon |
| 晚上 | wǎnshang | evening |
| 早 | zǎo | early; morning |
| 午 | wǔ | noon |
| 下 | xià | down; get off |
| 晚 | wǎn | evening; late |
| 起 | qǐ | get up |
| 床 | chuáng | bed |
| 起床 | qǐchuáng | get up |
| 吃 | chī | eat |
| 早饭 | zǎofàn | breakfast |
| 午饭 | wǔfàn | lunch |
| 晚饭 | wǎnfàn | dinner |
| 去 | qù | go |
| 上学 | shàngxué | go to school; attend school |
| 上课 | shàngkè | attend class |
| 放学 | fàngxué | classes are over |
| 下课 | xiàkè | finish class |
| 睡觉 | shuìjiào | sleep; go to bed |

### Key Sentence Patterns

**Describing daily routine:**
- 我早上X点起床。(I get up at X in the morning.)
- 我X点一刻去上学。(I go to school at quarter past X.)
- 我们X点半上课。(We start class at X:30.)
- 我们下午X点半放学。(We finish school at X:30 PM.)
- 我晚上X点半睡觉。(I go to bed at X:30 PM.)

**Question patterns:**
- 你早上几点起床？(What time do you get up?)
- 你几点吃早饭？(What time do you eat breakfast?)
- 你几点去上学？(What time do you go to school?)
- 你们下午几点放学？(What time do you finish school?)
- 你几点睡觉？(What time do you go to bed?)

### Key Text (Textbook p94)
```
我早上六点半起床。
我六点三刻吃早饭。
我七点一刻去上学。
我们八点上课。
我们下午三点半放学。
我晚上九点半睡觉。
```

### Workbook Exercises (types)
- Answer personal questions about daily routine
- Report back to class about partner's routine
- Listening and ticking correct times
- Writing sentences about daily schedule

## Lesson 12: Means of Transport (交通工具)

### Core Vocabulary

| 汉字 | 拼音 | English |
|---|---|---|
| 开 | kāi | open; drive |
| 车 | chē | vehicle; car |
| 开车 | kāichē | drive a car |
| 走 | zǒu | walk |
| 路 | lù | road |
| 走路 | zǒulù | walk |
| 坐 | zuò | sit; travel by (bus, train, etc.) |
| 校车 | xiàochē | school bus |
| 上班 | shàngbān | go to work |
| 下班 | xiàbān | finish work |
| 每 | měi | every |
| 每天 | měitiān | every day |
| 怎么 | zěnme | how |
| 火车 | huǒchē | train |
| 飞机 | fēijī | plane |
| 出租车 | chūzūchē | taxi |
| 公共汽车 | gōnggòng qìchē | public bus |
| 地铁 | dìtiě | subway |
| 校车 | xiàochē | school bus |

### Key Sentence Patterns

**Asking about transport:**
- 你怎么上学？(How do you go to school?)
- 你妈妈怎么上班？(How does your mum go to work?)

**Answering:**
- 我坐校车上学。(I go to school by school bus.)
- 我走路上学。(I walk to school.)
- 妈妈开车上班。(Mum drives to work.)
- 我每天坐公共汽车上学。(Every day I go to school by public bus.)

**Negation:**
- 我妈妈不工作。(My mum doesn't work.)

### Key Text (Textbook pp98-102, Text 2)
Dialogue about how family members commute:
```
我爸爸开车上班。
我妈妈走路上班。
我每天坐校车上学。
```
Extended with: 你妈妈呢？她坐地铁上班。你哥哥怎么上学？他坐公共汽车。

### Additional Vocabulary (Extra Words section)
- 火车 (huǒchē) — train
- 飞机 (fēijī) — plane
- 出租车 (chūzūchē) — taxi
- 电车 (diànchē) — tram
- 渡船 (dùchuán) — ferry
- 汽车 (qìchē) — car

### Grammar Notes
- 坐 + [vehicle] = travel by [vehicle]: 坐飞机 (by plane), 坐地铁 (by subway)
- 走路 = walk (no 坐 needed)
- 开车 = drive (a car)
- Time expressions typically come before or after the subject: 我每天坐校车上学 / 每天早上妈妈开车上班
- 每 + [time word] = every [time]: 每天 (every day), 每年 (every year)

### Phrase Pairs (Textbook p100)
Students learn directional pairs:
- 上课 (attend class) / 下课 (finish class)
- 上学 (go to school) / 放学 (classes are over)
- 上班 (go to work) / 下班 (finish work)

### Workbook Exercises (types)
- Question-answer matching
- Writing sentences describing how people commute
- Listening exercises
- Sentence reordering

### Test Coverage (from y7-tests/)
- **Test 4.3**: Translation sentences — "I go to school by plane every day at 7:00 am", "How does your mum go to work?", "Mum doesn't work.", "My dad gets off work at 5:30 pm."
- Time vocabulary tables (中→拼→英)
- Transport vocabulary

## Unit 4 Cross-Cutting Notes

### Skills Progression
1. Lesson 10 → Tell the time (foundation)
2. Lesson 11 → Use time to describe daily routine (builds on Lesson 10)
3. Lesson 12 → Describe transport + connect time + routine (synthesizes all three)

### Common Student Errors
- Using 二 instead of 两 for o'clock (两点 not 二点)
- Word order: 坐+vehicle+activity, not activity+坐+vehicle
- Forgetting 上/下 for directional pairs (上学, 上班, not just 学, 班)
- Mixing up 上午 and 早上; 下午 and 晚上
