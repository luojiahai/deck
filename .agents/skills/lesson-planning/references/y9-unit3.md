### Grammar Notes

**对……感兴趣 — be interested in**
对 marks the object of interest. Structure: Subject + 对 + topic + 感兴趣.
Common error: using the wrong preposition (do not say 我在历史很感兴趣).

**因为……所以…… — because…therefore…**
Both conjunctions can be used together, or either can appear alone. Avoid using 所以 without 因为 in written work.

**觉得 — feel/think that**
觉得 + adjective/clause: 我觉得汉语很难学。

**难/容易 + verb compounds**
难学 = difficult to learn; 难写 = difficult to write; 容易学 = easy to learn.

**门 — measure word for academic subjects/courses**
一门课, 十门课. Use 门 for courses, not 个.

**Verb + 得 + complement**
教得好 (teaches well), 写得好 (writes well). The complement describes the manner or result of the action.

**Approximate numbers with consecutive digits**
两三次, 四五个, 十五六岁. Two consecutive numbers placed together imply "approximately." (See Activity 10.)

### Grammar Notes

**Measure words for facilities**
- 个 is the default for most rooms and facilities: 一个图书馆, 一个礼堂, 一个操场
- 幢 (zhuàng) for entire building blocks: 五幢教学楼
- 场 (chǎng) appears inside compound words for open fields/courts: 篮球场, 足球场, 操场, 停车场 (no separate measure word needed — 场 is part of the word)

**层 = 楼 for floor levels**
三层 and 三楼 are interchangeable when referring to floor levels. Both appear in the textbook texts and should be taught together.

**Location word structure**
Location words (旁边, 隔壁, 中间, 左边, 右边) follow the reference point: X 在 Y 的 + location word.
Common error: "X 旁边在 Y" — wrong order; always X 在 Y 的旁边.

**游泳池 vs 游泳馆**
游泳池 (yóuyǒngchí) is the textbook term (p. 70). Test 3.2 uses 游泳馆 — this may be a workbook addition or teacher supplement. Teach 游泳池 as the primary term; flag 游泳馆 as a test variant students must also know.

### Grammar Notes

**一……就…… — as soon as X, immediately Y**
Structure: Subject + 一 + action1 + 就 + action2.
Example: 你一进校门就能看见图书馆。
Common error: placing 一……就…… at the end of the sentence — the structure must wrap the two actions.

**借 — borrow AND lend**
借 alone typically means "borrow" (我借你的铅笔).
借给 means "lend to" (借给我一支 = lend me one).
Context determines direction.

**还 (huán, 2nd tone) — return something borrowed**
Do not confuse with 还 (hái, 2nd tone) = also/still. Same character, same tone, different meaning — context distinguishes them. 我一会儿就还给你 = I'll return it to you soon.

**别 + verb — negative imperative**
别忘记 = don't forget. 别 is informal; 不要 is also acceptable.

**行 vs 可以**
行 = informal "OK/all right" (permission response or question).
可以 = more formal "may/can" (permission).
Both are used in Activity 9 translation exercises.

**Complement of result (Activity 4)**
Four categories introduced:
- 看见 / 听见 (perceive with senses)
- 看到 / 找到 / 买到 (achieve/attain)
- 吃光 / 喝光 / 卖光 (finish completely)
- 买好 / 做好 (finish successfully)

### Recurring exercise formats (across all three lessons)

- **Picture↔word matching** (12 items each lesson): subjects (incl. 美术), facilities, stationery (incl. 钢笔/毛笔)
- **Measure-word fill-from-box**: large drilled sets — L8 spans 个/件/双/门/张/架/辆/朵/把/幢/副/顶/套/节/位/条/头/间/家/种; L9 adds 块/本/盒. **幢 for buildings is heavily drilled.**
- **Character/radical work**: write radical meanings (disease/fire/claw/bow/ice/feeling/cliff/vehicle…), decompose & combine components (实→宀+头; 科→禾+斗), add-a-character to form phrases, word chains (滑冰→冰球→球鞋→鞋子)
- **Sentence-building from word groups** using unit patterns: 因为……所以…… / 对……(不)好 / 对……感兴趣 / 觉得 / 一……就…… / 难·容易学 / (没)有用 / 别 + complement of result
- **Antonym (opposite) pairs**: 难↔容易, 没用↔有用, 难看↔漂亮, 还↔借, 出↔进, 卷↔直
- **Reading comprehension + cloze (box-fill)**: school reports, school descriptions, over-stocked stationery; with T/F and short-answer questions
- **Internet research + type-a-passage on computer**, then a **writing task (80–100 words)** every lesson

### Lesson-specific highlights

- **L7**: timetable fill + cloze (星期五课程表); evaluate subjects (有意思/没意思/有用/没用, 对……感兴趣); approximate/consecutive-number practice (mirrors textbook 两三次).

- **L9**: personal-answer questions with 一……就……; dictionary guess-then-check pairs (饭盒/盒饭, 直尺/三角尺); sentence-final particles 吗/呢/吧; 正在 past-action questions; **social dialogues** (对不起 / 谢谢你 / 请坐 / 你打错电话了); verb fill-from-box (借/还/穿/带/戴/教/写/滑/买/踢); C→E translation with 别; borrow/lend & "你怎么办" problem-solving.

### Common Student Errors

| Error type | Wrong | Correct |
|-----------|-------|---------|
| 对……感兴趣 — missing 对 | 我在历史很感兴趣 | 我对历史很感兴趣 |
| 因为……所以…… — 所以 without 因为 | 所以我喜欢化学 | 因为……所以我喜欢化学 |
| Location word order | X 旁边在 Y | X 在 Y 的旁边 |
| 还 (huán) vs 还 (hái) | (using 还 hái when meaning "return") | 还 (huán) = return; 还 (hái) = also/still |
| Overuse of 个 for all measure words | 一个铅笔, 一个教学楼 | 一支铅笔, 一幢教学楼 |
| 一……就…… — wrong word order | 我进校门就看见一…… | 我一进校门就能看见 |
| 游泳池 vs 游泳馆 | (using one without knowing the other) | 游泳池 = textbook term; 游泳馆 = test variant |
