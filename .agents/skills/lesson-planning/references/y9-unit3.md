### Key Sentence Patterns

**Listing subjects:**
- 我今年学十门课：英文、汉语、音乐、数学、电脑、物理、化学、生物、地理和历史。

**Expressing opinions about subjects:**
- 我喜欢上历史课，因为老师对我很好，我对历史很感兴趣。
- 我也喜欢学汉语，因为我觉得汉语很有用。
- 但是汉语很难学，因为汉字不容易写。
- 我觉得物理很难学。
- 我觉得地理不难学。

**因为…所以… (because…therefore…):**
- 因为我对历史很感兴趣，所以我喜欢上历史课。

**Asking about subjects:**
- 你最喜欢什么科目？
- 你觉得什么科目最难/最容易？
- 你为什么喜欢X课？

### Key Text (Textbook p64)
```
我今年学十门课：英文、汉语、音乐、数学、电脑、物理、化学、生物、地理和历史。我喜欢上历史课，因为老师对我很好，我对历史很感兴趣。我也喜欢学汉语，因为我觉得汉语很有用。但是汉语很难学，因为汉字不容易写。
```

### Grammar Notes
- 对…感兴趣 = be interested in (对 marks the target of interest)
- 因为…所以… = because…therefore… (both can be used together, or separately)
- 觉得 + adjective = feel/think that something is…
- 难学 = difficult to learn; 难写 = difficult to write
- 门 (mén) is the measure word for subjects/courses

### Test Coverage (from y9-tests/)
- **Test 3.1**: Answer questions in Chinese about school subjects; vocabulary table: subject names, adjectives (难, 容易, 有用), time expressions, opinion words

### Key Sentence Patterns

**Introducing school facilities:**
- 我来介绍一下我们学校。(Let me introduce our school.)
- 我们学校有X个教学楼、一个图书馆、一个礼堂、一个操场和一个体育馆。
- 还有两个篮球场和一个游泳池。

**Describing location:**
- X在Y楼。(X is in Building Y.)
- X在Y楼楼上/楼下。(X is upstairs/downstairs from Y.)
- X在Y旁边/隔壁。(X is next to Y.)

### Key Text (Textbook ~p74)
```
我来介绍一下我们学校。我在英文学校上学。我们学校有一个教学楼、一个图书馆、一个礼堂、一个操场和一个体育馆。还有两个篮球场和一个游泳池，还有一个停车场。
```

### Grammar Notes
- 个 is the default measure word for rooms/facilities
- 场 (chǎng) for open fields/courts
- 楼 (lóu) for buildings; 层 (céng) for floor levels (三层 = 3rd floor)
- Location words (旁边, 隔壁, 中间) follow the noun they modify

### Key Sentence Patterns

**Borrowing and lending:**
- 借我X块钱，行吗？(Can I borrow X dollars?)
- 我忘记带钱包了。我明天还给你。(I forgot my wallet. I'll return it tomorrow.)
- 别忘记明天还给我。(Don't forget to return it to me tomorrow.)

**一…就… (as soon as):**
- 你一进校门就能看见图书馆。(As soon as you enter the school gate, you can see the library.)

**Permission and capability:**
- 我可以看看这条项链吗？(May I look at this necklace?)
- 行吗？/ 可以吗？(Is that OK? / May I?)
- 不行。/ 可以。(No. / Yes, you may.)

### Key Text (Textbook ~p84)
```
文具店里卖铅笔、橡皮、卷笔刀、练习本、日记本和文具盒。除了这些，文具店还卖小说、课本、卡片等。文具店在五号教学楼里，在一楼。你一进校门就能看见。
```

### Grammar Notes
- 一…就… = as soon as X, then Y (connects two actions in quick succession)
- 借 = can mean both "borrow" AND "lend" depending on context
- 还 (huán) = return (something borrowed); NOT the same as 还 (hái = also/still)
- 别 + verb = don't (negative imperative)
- 行 = "OK" / "all right" (informal permission)

## Unit 3 Cross-Cutting Notes

### Skills Progression
1. Lesson 7 → School subjects + expressing opinions + 因为…所以…
2. Lesson 8 → School facilities + location words + describing places
3. Lesson 9 → Stationery + borrowing/lending + 一…就… + permission

### Common Student Errors
- Confusing 还 (huán = return) with 还 (hái = also/still)
- Word order with location phrases: "X 在 Y 旁边" not "X 旁边在 Y"
- 对…感兴趣: forgetting 对 or using wrong preposition
- Mixing up measure words for different items (支 for pens, 本 for books, 把 for rulers, etc.)
- 一…就… word order: subject + 一 + action1 + 就 + action2

### Textbook Activities & Radicals

Like all 轻松学中文 units, Unit 3 includes:
- **Radicals**: Each lesson has a "Learn the radicals" section. Identify which radicals appear and teach them alongside relevant characters (e.g. 辶(walk) → 进, 过; 扌(hand) → 把).
- **Speaking practice**: Dialogue and model-sentence exercises throughout.
- **Listening exercises**: Listen and tick/circle/answer.
- **Pinyin exercises**: Pinyin discrimination and tonal mark practice (important even at Y9 level for new vocabulary).
- **Group/inquiry activities**: Category grouping, measure word matching.

When planning lessons for this unit, check the textbook pages for the specific radicals, pinyin pairs, and activity formats — cover them all across the lesson sequence.

### Key Measure Words Introduced/Reinforced
- 门 (mén) — subjects, courses
- 个 (gè) — rooms, facilities (default)
- 层 (céng) — floors of a building
- 本 (běn) — books, notebooks
- 支 (zhī) — pens, pencils
- 把 (bǎ) — rulers, things with handles
- 条 (tiáo) — long, thin items
