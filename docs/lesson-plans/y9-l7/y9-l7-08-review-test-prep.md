# Review + Test 3.1 Prep — Subjects of Study 科目

**Year Level:** Year 9
**Unit:** Unit 3 — School Life
**Lesson:** Lesson 7 — Subjects of Study 科目 (Sequence 8 of 8)
**Duration:** 50 minutes

## Learning Intention
We are learning to consolidate all Lesson 7 vocabulary and sentence patterns, and to prepare accurate spoken and written responses for Test 3.1.

## Success Criteria
- 我能翻译Lesson 7所有测验词汇（中文↔英文） (I can translate all Test 3.1 vocabulary in both directions)
- 我能用"对……感兴趣"、"觉得"、"因为……所以……"和"verb+得"造出完整的句子 (I can produce complete sentences using all four key patterns)
- 我能用中文回答：你今年学几门课？你最喜欢哪个科目？你每天作业要做多长时间？(I can answer all three Test 3.1 answer-in-Chinese questions)

## Materials Needed
- Projector (display vocabulary grid, patterns, quiz questions, sentence frames, exit ticket)
- CD player — Track 28 (Act. 12, p. 68)
- Students: exercise books and pens
- Teacher: scoreboard drawn on whiteboard corner for team quiz

---

## Lesson Sequence

### Review (0–8 min)

**What:** Rapid recall of core Lesson 7 vocabulary and all four key sentence patterns.

**How:** Project 10 quick-fire prompts one at a time. For vocabulary prompts (items 1–7), display the Chinese character(s) and cold-call one student, then confirm with a choral repeat. For sentence-pattern prompts (items 8–10), display the sentence starter and cold-call a student to complete it; class echoes the full sentence chorally.

Quick-fire sequence:
1. 物理 → *physics*
2. 生物 → *biology*
3. 化学 → *chemistry*
4. 有用 → *useful*
5. 觉得 → *feel / think (that)*
6. 测验 → *test / quiz*
7. 学期 → *semester / term*
8. Starter: 我对历史很…… → *感兴趣*（complete + choral echo of full sentence）
9. Starter: 因为我觉得汉语很难学，所以…… → *elicit student completion*（choral echo）
10. Starter: 老师教得…… → *好 / 不好*（cold call + choral echo）

OTR: cold call (individual) + choral echo (whole class) on every item.

**Connection:** 今天是第八节课。我们复习所有的内容，然后练习考试会考的问题。这节课结束后你们就准备好考测验3.1了。

Point out that the three SC map directly onto what Test 3.1 will assess. Leave SC visible on a corner of the board (or projected) for the remainder of the lesson.

#### Key Sentence Pattern Summary

Display all eight patterns with one modelled example each. Students copy into exercise books.

| # | Pattern | Modelled example |
|---|---|---|
| 1 | 门 — measure word for courses | 我今年学十门课。 |
| 2 | 觉得 + adjective/clause | 我觉得物理很难学。 |
| 3 | 对……感兴趣 | 我对历史很感兴趣。 |
| 4 | 因为……所以…… | 因为我觉得化学很有意思，所以我喜欢化学。 |
| 5 | Verb + 得 + complement | 老师教得很好。 |
| 6 | 多长时间 (duration question) | 你每天功课要做多长时间？ |
| 7 | Approximate numbers (consecutive digits) | 每个学期考两三次。 |
| 8 | 经常 / 常常 | 我们每个月经常有测验。 |

**Delivery (14–16 min, 2 min):** Flash each row one at a time. Teacher reads the example aloud; class repeats chorally. Point out Pattern 4 common error: never use 所以 alone without 因为 in written work.

#### Text / Dialogue

**Act. 12 (p. 68, CD Track 28) — Listen and Tick (16–20 min, 4 min)**

Students draw a 6-row answer grid in their exercise books (label rows 1–6). Play CD Track 28 once straight through. Students tick the correct answer for each of the 6 items about a student's school life.

After playback, teacher reveals answers on projector (or reads them aloud), students self-mark.

**The three Test 3.1 answer-in-Chinese questions** — display on projector for 30 seconds:
1. 你今年学几门课？
2. 你最喜欢哪个科目？
3. 你每天作业要做多长时间？

Teacher models one sample answer for each (cold-call a strong student for items 2 and 3 before modelling). These same questions will appear in the Team Quiz and the written practice.

### Flexible Practice (20–43 min)

#### Activity 1: 自我介绍 (Act. 15, p. 69) — We Do (20–30 min, 10 min)

**What:** Students prepare and deliver an extended self-introduction covering: subjects studied this year, opinions on those subjects, favourite subject (with reason), and least-favourite subject (with reason). Based on textbook Activity 15.

**How:**
1. Display the prompt framework on projector (2 min planning):
   - 你今年学几门课？学什么？
   - 你最喜欢哪个科目？为什么？
   - 你觉得哪个科目最难/最容易？
   - 你最不喜欢哪个科目？为什么？
2. Students write 5–6 sentences in exercise books (3 min).
3. Turn-and-talk: A speaks for ~45 seconds; B listens then asks at least one 为什么 follow-up question; swap (4 min total).
4. 2–3 volunteer pairs share one sentence each with the class (1 min).

- **Scaffold:** Keep this sentence frame projected throughout: 我今年学______门课：______。我最喜欢______，因为______。我觉得______很______，因为______。我对______很感兴趣。我最不喜欢______，因为我觉得______。
- **Extension:** Add homework duration using 多长时间, test frequency using approximate numbers (每个月测验两三次), and verb+得 to describe a teacher (老师教得很好/有点无聊). Remove the sentence frame before they speak.
- **OTR:** Turn-and-talk (every student speaks); volunteer share-out (cold call if no volunteers).

---

#### Game: 科目知识大挑战 — Projector Team Quiz (30–40 min, 10 min)

**What:** A four-round team quiz reviewing all Lesson 7 vocabulary and patterns. No devices — questions displayed on projector, teams answer aloud or hold up exercise books.

**Setup:** Divide class into 4 teams. Teacher draws a scoreboard on the whiteboard corner (Team A / B / C / D). Teams raise hands to buzz in; teacher cold-calls if no hands after 5 seconds. One point per correct answer; bonus point for an extended answer where indicated.

**Round 1 — 词汇闪卡 Vocabulary Flash (5 questions, ~1 min):**
Display each Chinese word; first team to call out the correct English wins the point.
1. 有意思 → *interesting*
2. 学期 → *semester / term*
3. 大概 → *approximately / about*
4. 没意思 → *boring / not interesting*
5. 两次 → *twice / two times*

**Round 2 — 英译中 English → Chinese (5 questions, ~2 min):**
Display the English; teams write the Chinese in exercise books, hold up — teacher picks one team to read aloud. (Hold-up-book OTR — every student must write.)
1. *be interested in* → 对……感兴趣
2. *difficult* → 难
3. *often / regularly* → 经常
4. *break time* → 课间休息
5. *examination* → 考试

**Round 3 — 句子挑战 Sentence Patterns (3 questions, ~3 min):**
Display an incomplete or mistaken sentence; teams confer for 10 seconds, then one team representative answers.
1. Complete: 因为我觉得历史很有意思，______我喜欢历史。 → *所以*
2. Correct the error: 我在数学很感兴趣。 → *我对数学很感兴趣。*
3. Complete using verb+得: 我们的数学老师______。 → *open answer — must use 教得 + complement, e.g. 教得很好*

**Round 4 — 用中文回答 Answer in Chinese (3 questions, ~4 min):**
Display each question; any team member answers in a full sentence. Bonus point for adding a reason or extra detail.
1. 你今年学几门课？ (1 pt base + 1 bonus if lists subjects using 门)
2. 你最喜欢哪个科目？ (1 pt base + 1 bonus for 因为……所以……)
3. 你每天作业要做多长时间？ (1 pt base + 1 bonus for using 大概 or 差不多)

- **Scaffold:** For Round 3, allow teams 10 seconds to confer before answering (reduces pressure for less confident students).
- **Extension:** The bonus-point system built into Round 4 naturally pushes extension students to produce more complex answers without creating a two-tier game.

---

#### Activity 2: 笔试练习 Written Practice (40–43 min, 3 min)

**What:** Short independent written task in the Test 3.1 answer-in-Chinese format.

**How:** Display all three questions on projector. Students write full-sentence answers in exercise books independently. No talking.

1. 你今年学几门课？
2. 你最喜欢哪个科目？
3. 你每天作业要做多长时间？

- **Scaffold:** Keep sentence starters projected below the questions: 我今年学______门课。/ 我最喜欢______，因为……/ 我每天作业大概要做______。
- **Extension:** Write a fourth sentence of their own choosing that uses 经常 or 常常 (e.g. 我们每个学期经常有考试。/ 我常常在放学后做功课。).

---

### Plenary (43–50 min)

- **Revisit Success Criteria:** Display the three SC on projector. Teacher reads each one in Chinese; students rate themselves silently: 👍 ready / ✋ getting there / 👎 need more practice. Quick show of hands for each rating on each criterion — teacher notes which SC has the most 👎 responses (signals what to address before Test 3.1).

- **Self-assessment:** Students write a single Chinese sentence at the bottom of today's page: 我最需要复习______，因为______。Teacher cold-calls 1–2 students to share.

- **Exit Ticket:** Display on projector:
  > 用中文写两三句话：你最喜欢哪个科目？为什么？
  >
  > (Write 2–3 sentences in Chinese: Which subject do you like most? Why?)

  Students write in exercise books. Teacher circulates and reads over shoulders — look for correct use of 觉得, 对……感兴趣, or 因为……所以……. This mirrors the exact format of Test 3.1 Question 2.
