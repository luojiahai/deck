# Time (v2) — Lesson 1: 现在几点? + O'clock (点)

**Year Level:** Year 7
**Unit:** Unit 4 — Time, Daily Routine, Transport (轻松学中文 1)
**Lesson:** Lesson 10 — Time (时间) · v2 sequence (communicative-first)
**Duration:** 50 minutes

> **v2 note:** This sequence leads with the *question* 现在几点? from lesson one so that every time-form is learned as an answer to a real exchange, not an isolated drill. v1 is left untouched.

## Learning Intention
We are learning to ask and answer what o'clock it is in Chinese.

## Success Criteria
- I can ask the time using 现在几点? (用中文问时间)
- I can say o'clock times using 点 (用「点」说整点)
- I can use 两点 (not 二点) for 2:00 (区分「两」和「二」)

## Materials Needed
- Projector/TV (clock-face images, sentence frames)
- A4 paper, one sheet per student (folded into 8 boxes for written responses — replaces mini whiteboards)
- Textbook 轻松学中文 1 (pp.70–73, 77)

---

## Lesson Sequence

### Review (0–8 min)
**What:** Numbers 1–12 in Chinese + a quick burst of prior-unit facts (p.72 Exercise 3 group questions) to settle the class and surface number vocabulary.
**How:** Project numbers 1–12 as digits in random order; class chorally says each in Chinese. Then fire 4 quick p.72 questions (你叫什么名字? 你家有几口人? 你今年上几年级? 你是哪国人?) — students jot keyword answers in box 1 of their folded paper, cold-call 2 per question.
**Connection:** "You already know your numbers — today we use them to tell the time, starting with the question everyone asks: 现在几点?"

### Learning Intention & Success Criteria (8–10 min)
Display all three success criteria on the projector and keep them visible all lesson. Read each aloud; class echoes the Chinese chunk (现在几点? / 点 / 两点).

### I Do (10–20 min)
#### New Vocabulary
| 汉字 | 拼音 | English |
|---|---|---|
| 现在 | xiàn zài | now |
| 几点 | jǐ diǎn | what time |
| 点 | diǎn | o'clock |
| 两 | liǎng | two (for counting / time) |

#### Key Sentence Pattern
**现在几点? → 现在 X 点。**
- Model with projected clock: 现在三点。/ 现在十点。
- Think-aloud for 2:00: "Careful — it is 两点, NOT 二点. When we tell the time, 2 is always 两."
- One silly example: project a clock at 12:00 with a cartoon — "现在十二点……午饭时间!" (wait for reaction).

#### CFU Checkpoints
- Thumbs up if 现在几点? means "What time is it now?"
- Project 7:00 — turn and tell your partner: 现在七点。
- Error catch: "现在二点 — what's wrong?" (should be 两点).

#### Text / Dialogue
Mini-exchange drilled chorally (half class A, half class B), then swap:
> A: 现在几点? B: 现在 ___ 点。

### Flexible Practice (20–43 min) — Mixed
#### Activity 1: Pair Q&A clock flash — We Do (7 min)
Project a clock face; partner A asks 现在几点?, partner B answers 现在 X 点。Switch every clock. Run 6–8 o'clock times (include 2:00 to force 两点).
- **Scaffold:** Sentence frame stays on screen; word bank of number characters 一–十二 visible.
- **Extension:** B adds a follow-up greeting matching the time (早上好 / 晚上好) or says the time without looking at the frame.
- **OTR:** Every student speaks each round (pair simultaneous talk).

#### Activity 2: Write the time — You Do (8 min)
Project 6 clocks numbered 1–6. Students write each as 现在 X 点。in boxes 2–7 of their folded paper. Then read 7 everyday-object clocks aloud from **textbook p.71 Exercise 1**.
- **Scaffold:** Provide the first answer as a model; allow pinyin above characters.
- **Extension:** Write characters only (no pinyin) and add 现在 to each full sentence.
- **CFU:** Hold up paper for box 3 and box 5 — quick scan for 两点 errors.

#### Game: Arms-as-Clock-Hands race (6 min) — adapts textbook p.73 Exercise 6
No materials. Two volunteers stand at the front and use their arms as hour/minute hands. Teacher (or a student caller) says an o'clock time in Chinese (e.g. 现在八点); the two race to make their arms show it; class judges 对 / 不对.
- **Setup needed:** None — projector optional to show the target as confirmation.

### Plenary (43–50 min)
- **Revisit Success Criteria:** Student volunteer reads each "I can…" aloud; class shows thumbs.
- **Self-assessment:** 👍 / 😐 / 👎 on each criterion (show of hands).
- **Exit ticket:** On the last box of their paper, write the answer to 现在几点? for a projected 9:00 clock (现在九点). Collected at the door.
- **Preview:** "Next lesson we get more precise — minutes (分), and the tricky little 零."
