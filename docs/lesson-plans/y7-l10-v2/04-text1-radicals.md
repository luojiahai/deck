# Time (v2) — Lesson 4: Text 1 Reading + Radicals (土 又 刀)

**Year Level:** Year 7
**Unit:** Unit 4 — Time, Daily Routine, Transport (轻松学中文 1)
**Lesson:** Lesson 10 — Time (时间) · v2 sequence (communicative-first)
**Duration:** 50 minutes

## Learning Intention
We are learning to read Text 1 fluently and to recognise three time-unit radicals.

## Success Criteria
- I can read Text 1's six clock times aloud (流利朗读课文一的六个时间)
- I can recognise the radicals 土, 又, 刀 and name their meaning (认识部首土/又/刀)
- I can build a phrase by adding the missing character (加字组词)

## Materials Needed
- Projector/TV (Text 1, radical cards, Exercise 8 prompts)
- A4 paper + pen, one per student
- Textbook 轻松学中文 1 (pp.70, 73, 75)

---

## Lesson Sequence

### Review (0–8 min)
**What:** All time-forms so far (点 / 分 / 零 / 刻 / 半).
**How:** Rapid choral: project 6 mixed clocks (one o'clock, one with 零, one quarter, one half, two free); class reads each. Then "Correct the Teacher" — project 二点半 and 六点五分; students call the fix (两点半 / 六点零五分).
**Connection:** "You can say every kind of time — now let's read them all together in the textbook's own Text 1, and look at how the characters are built."

### Learning Intention & Success Criteria (8–10 min)
Project; read aloud; class echoes radical names 土/又/刀. Keep visible.

### I Do (10–20 min)
#### Reading focus — Text 1 (textbook p.70, CD T45)
Six clock times: 十点 / 六点零五分 / 七点十分 / 九点一刻 / 两点半 / 十一点三刻.
- Teacher reads aloud, students follow in the book.
- Choral read (whole class), then half-and-half (boys/girls or rows).
- 2–3 comprehension questions: "Which clock shows a quarter past?" "Which uses 零?" "Read me the 11:45 one."

#### New Radicals (textbook p.73 Exercise 5 — first three)
| 部首 | Meaning | Example character |
|---|---|---|
| 土 | soil | 地 |
| 又 | again | 友 |
| 刀 | knife | 色 |

Show each radical, its meaning, and the example character. Point out the radical inside the example.

#### CFU Checkpoints
- "Point to the 又 in 友." (thumbs/point)
- "What does 土 mean?" (soil)
- Quick read-check: cold-call 2 students to read one Text 1 line each.

### Flexible Practice (20–43 min) — Mixed
#### Activity 1: Text 1 partner read + comprehension — We Do (8 min)
Partners take turns reading the six times; then answer 3 projected questions in their paper boxes (e.g. "把九点一刻写成数字" → 9:15).
- **Scaffold:** Pinyin layer visible above Text 1; allow English for comprehension answers.
- **Extension:** Cover the pinyin; answer comprehension in Chinese; convert all six to digits.
- **OTR:** Both partners read aloud; all write the answers.

#### Activity 2: Add the missing character — You Do (8 min) — textbook p.75 Exercise 8
Project phrase prompts with a blank (e.g. 明__, 多__, 几__, __在). Students write the completed phrase in their boxes; include 现__ → 现在 and 几__ → 几点 to tie back to time.
- **Scaffold:** First three done as models; word bank available.
- **Extension:** Do all items, then write the pinyin for two of the completed words.
- **CFU:** Hold up the 现在 / 几点 answers for a scan.

#### Game: Radical Slap (projected) (6 min)
Project the three radicals 土 / 又 / 刀 in three large boxes. Teacher calls a meaning ("knife!" / "soil!") or an example character ("友!"); two volunteers race to slap the correct radical box on the screen/board; class verifies.
- **Setup needed:** Projector with the three radicals displayed; no printing.

### Plenary (43–50 min)
- **Revisit Success Criteria:** Volunteer reads; thumbs per criterion.
- **Self-assessment:** 👍 / 😐 / 👎.
- **Exit ticket:** Write the meaning of 土, 又, 刀 (soil / again / knife) and read aloud one Text 1 time to the teacher on the way out.
- **Preview:** "Next: the cool one — saying the time *backwards* with 差 (minutes-to), plus 你的表几点了?"
