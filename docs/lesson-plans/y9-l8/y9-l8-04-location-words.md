# Year 9 — 学校设施 (Seq. 4 of 8): Location Words 旁边/隔壁/中间/左右

**Year Level:** Year 9
**Unit:** Unit 3 — School Life
**Lesson:** Lesson 8 — School Facilities 学校设施 (Sequence 4 of 8)
**Duration:** 50 minutes

## Learning Intention

We are learning to say where something is using 旁边、隔壁、中间、左边、右边 in the correct word order.

## Success Criteria

- I can use X 在 Y 的 + location word in the correct order (能正确使用"X在Y的……"的语序)
- I can find and correct the common word-order error with location words (能找出并改正位置词的语序错误)
- I can describe where a room or facility is using a complete sentence (能用完整句子说明教室或设施的位置)

## Materials Needed

- Projector (all visuals displayed and controlled from projector only)
- Students' exercise books and pens
- No printed cards, no mini whiteboards, no student devices required

---

### Review (0–8 min)

**What:** Room vocabulary and 层=楼 from Lesson 3, plus the location words students met receptively in Text 2.

**How:**

*Part A — Room vocabulary flash (3 min):* Project 6 room words from Lesson 3 (教室, 实验室, 厕所, 办公室, 校医室, 音乐室) one at a time. Class chorally names each in Chinese, cold-call one student per word for the English.

*Part B — Location word recognition (5 min):* Redisplay the four Text 2 sentences containing location words (法语教室的**旁边**; 音乐室的**隔壁**; 教师办公室和校医室的**中间**; 男厕所在**左边**，女厕所在**右边**). Cold call students to translate each bolded word into English.

**Connection:** "上节课我们在课文里看到了这些位置词，但是还没有自己练习用它们说句子。今天我们要专门练习——而且要小心一个很多同学会犯的错误。"

---

### Learning Intention & Success Criteria (8–10 min)

Project the Learning Intention and all three Success Criteria. Teacher reads the Learning Intention aloud; a student volunteer reads each Success Criterion. Leave visible in a corner of the screen for the rest of the lesson.

---

### I Do (10–20 min)

#### Vocabulary Recap (already met receptively — confirm and anchor)

| 汉字 | 拼音 | English |
|---|---|---|
| 旁边 | pángbiān | beside; next to |
| 隔壁 | gébì | next door |
| 中间 | zhōngjiān | between; centre |
| 左边 | zuǒbiān | left side |
| 右边 | yòubiān | right side |

Choral read all five once; move straight to the grammar structure, since these words are already familiar from Lesson 3.

#### Key Sentence Pattern

**Structure:** X 在 Y 的 + [location word]。

**Modelled examples (project each one, teacher reads aloud, class repeats):**

1. 汉语教室在法语教室的旁边。
2. 物理实验室在音乐室的隔壁。
3. 校长办公室在教师办公室和校医室的中间。
4. (Silly example — wait for reaction:) 校长办公室在游泳池的旁边！

**Grammar note — explicit teaching:**

Project the rule: "位置词跟在参照物后面：X 在 Y 的 + 位置词。" Then project the common error side by side:

❌ 图书馆旁边在礼堂。
✅ 图书馆在礼堂的旁边。

"位置词永远跟在'的'后面，不能放在句子最前面。" Point out this is the single most common mistake with this pattern — students must check word order every time.

#### CFU Checkpoints

- **(Gesture)** Project the two sentences above (❌ / ✅). "哪个对？举手指。" Every student points.
- **(Cold call)** "Alex, 用'隔壁'说一句话，说说体育馆在哪儿。" → accept any grammatically correct sentence using X在Y的隔壁.
- **(Choral — error correction)** Project: 餐厅左边在图书馆。 "这句话有错误。大家一起说正确的！" → class calls out "餐厅在图书馆的左边！"

---

### Flexible Practice (20–43 min)

#### Activity 1: 改错句 Guided Error Correction — We Do (7 min) | 20–27 min

**What:** Guided practice fixing word-order errors with location words, moving from teacher-modelled to student-produced.

**How:**
- Project 4 sentences with the word-order error, one at a time. Teacher models fixing the first two aloud, thinking through the rule each time: "'旁边'在最前面——错！应该是'X在Y的旁边'。"
  1. 图书馆旁边在礼堂。→ 图书馆在礼堂的旁边。
  2. 实验室隔壁在音乐室。→ 实验室在音乐室的隔壁。
- Turn-and-talk (2 min): partners fix the remaining two together.
  3. 校长办公室中间在教师办公室和校医室。
  4. 男厕所左边在女厕所。
- Cold call two pairs to share their corrected sentences; class echoes the correct version chorally.

**OTR 1 (every student responds):** Turn-and-talk requires every student to attempt both remaining corrections before cold call.

- **Scaffold:** Keep the grammar rule (X 在 Y 的 + 位置词) projected throughout as a checklist.
- **Extension:** After fixing each sentence, fluent speakers must also state which reference point (Y) the location word is describing, out loud: "参照物是'礼堂'。"

---

#### Activity 2: 位置词造句 — You Do, Writing (9 min) | 27–36 min

**What:** Students write five original sentences, one using each location word, describing where rooms and facilities are in their own school (real or invented), then swap for peer-checking.

**How:**
- Project a word bank combining Lesson 1–3 vocabulary: 图书馆, 礼堂, 操场, 体育馆, 篮球场, 足球场, 餐厅, 游泳池, 停车场, 教室, 实验室, 厕所, 办公室, 音乐室, 校医室.
- Teacher models one example live: "我们学校的图书馆在体育馆的旁边。"
- Individual writing (5 min): students write 5 sentences in exercise books, one per location word, each following X 在 Y 的 + location word exactly.
- Peer-check (2 min): students swap books with a neighbour; the neighbour circles any word-order errors in pencil without fixing them.
- Students self-correct any circled sentences.

**OTR 2 (every student responds):** All students write independently; peer-check requires every student to read and mark a partner's work.

- **Scaffold:** Provide a gap-fill frame for each sentence: 我们学校的______在______的______。 with the location word already given for each of the 5 lines.
- **Extension:** Students must write all 5 sentences unaided (no gap-fill), and add one sentence using 中间 with two named reference points, matching the Text 2 pattern exactly (X 在 Y 和 Z 的中间).

---

#### Game: 改错句接力赛 Error-Fixing Relay (7 min) | 36–43 min

**How to play:**

1. Divide the class into 3–4 teams.
2. Project one word-order-error sentence at a time (new sentences, not reused from Activity 1), e.g.: 操场旁边在游泳池。/ 校医室隔壁在办公室。/ 停车场左边在教学楼。
3. The first student from the next team in rotation must say the corrected sentence aloud within 10 seconds. Correct = 1 point for their team; if wrong or too slow, the next team may steal.
4. Play 6–8 rounds, keeping score on the whiteboard corner or projector.

**What language it practises:** Rapid application of the X 在 Y 的 + 位置词 rule under time pressure; reinforces error recognition.

**Setup needed:** Projector only — prepare a slide with 6–8 error sentences in advance; whiteboard corner or projected scoreboard for tallying points.

- **Scaffold:** Teacher may repeat the sentence once if a student mishears it before the 10-second timer starts.
- **Extension:** Winning students must add the reference-point identification for a bonus point: "参照物是'游泳池'。"

---

### Plenary (43–50 min)

- **Revisit Success Criteria:** Project the three Success Criteria. Teacher reads the first; student volunteers read the second and third. "今天我们做到了吗？"

- **Self-assessment:** Students hold up fingers — 3 = 完全可以, 2 = 差不多, 1 = 还不行 — for each criterion. Note any students showing 1 finger on criterion 2 (error correction) for the next lesson's review.

- **Exit ticket (exercise books, 2 min):** Students write ONE sentence describing where their favourite school facility is, using a location word: 我最喜欢的地方是______，它在______的______。

- **Preview:** "下一课，我们要学更多位置词——上面、下面、前面、后面、对面、楼上、楼下——用来描述教室里的东西和整栋楼的布局。"
